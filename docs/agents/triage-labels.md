# Triage labels

Engineering skills 使用五个 canonical triage roles。当前仓库直接使用默认名称,不做二次映射。

| Skill label | Tracker label | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 需要 maintainer 评估 |
| `needs-info` | `needs-info` | 等待报告者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 已充分定义,可由 AFK agent 执行 |
| `ready-for-human` | `ready-for-human` | 需要 human 实施 |
| `wontfix` | `wontfix` | 不会进入实施 |

Skill 提到某个 triage role 时,使用表中的 tracker label。若仓库以后改用其他标签词汇,只修改右侧映射。
